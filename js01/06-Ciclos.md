# Ciclos

### For

```js   
function imprimirAzul4(){

  for(var i = 0; i < 4; i++) {
     console.log("Azul")
  }
}

// Con parametro
function imprimirAzul(n){
  for(let i=0; i < n;i++){
    console.log('Azul')
  }
}
```
